dotemacs
========

My personal .emacs file.

Install
-------

Just create a symbolic link from ~/.emacs to dotemacs/dot_emacs.

Requirements
------------

Emacs packages:

1. yasnippet
2. jedi

Python packages:

1. Jedi: http://tkf.github.com/emacs-jedi/
1.1 python-epc (sudo pip install epc)
1.2 Jedi (sudo pip install jedi)
